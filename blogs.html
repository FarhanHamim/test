Q1: How can you select elements in the DOM?

There are multiple ways to select elements: use document.getElementById for unique elements, getElementsByClassName or getElementsByTagName for groups, and querySelector or querySelectorAll for CSS-style selections. You can also traverse the DOM using properties like parentElement, children, and nextElementSibling.

Q3: What is event delegation?

Event delegation is a technique where a single event listener is attached to a parent element to handle events for its child elements. This approach leverages event bubbling, making the code more efficient and easier to manage.

Q4: What is event bubbling?

Event bubbling is a process where an event starts at the target element and propagates up through its ancestors in the DOM tree. This allows parent elements to handle events triggered by their children.

Q5: How do you create, add, and remove elements in JavaScript?

Use document.createElement() to create new elements, appendChild() or insertBefore() to add them to the DOM, and remove() or removeChild() to delete them. These methods enable dynamic updates to web pages.
